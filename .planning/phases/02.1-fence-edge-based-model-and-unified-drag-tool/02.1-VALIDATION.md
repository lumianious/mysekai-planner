## Sampling Rate

- **After every task commit:** Run quick command (only edge-related tests + tsc on changed files)
- **After every plan wave:** Run full suite command
- **Before `/gsd:verify-work`:** Full suite must be green AND manual smoke (edges visually between tiles, coexist with roads) confirmed
- **Max feedback latency:** 15 seconds for per-task; 45 seconds for per-wave
