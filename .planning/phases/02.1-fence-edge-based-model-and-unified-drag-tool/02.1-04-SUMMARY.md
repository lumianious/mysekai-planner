## Coordinate Math Reference

```
FENCE_THICKNESS = 6
TILE_SIZE = 32

Horizontal edge (x, y, 'h') — top border of tile (x,y):
  render.x = x * 32
  render.y = y * 32 - 3      (centered on line)
  width    = 32
  height   = 6

Vertical edge (x, y, 'v') — left border of tile (x,y):
  render.x = x * 32 - 3      (centered on line)
  render.y = y * 32
  width    = 6
  height   = 32
```

## Decisions Made

- **fixtureMap as prop:** Matches existing GroundLayer/FurnitureLayer pattern. FenceLayer receives `fixtureMap` from EditorCanvas, not from a hook or context.
- **No performance optimization:** No `useMemo` or `useEffect` — the render loop iterating `Object.values(placedEdges)` is cheap at the expected scale (<=200 edges). Research pitfall 5 confirmed.
- **FENCE_THICKNESS=6:** 6px provides a visible but proportional fence at TILE_SIZE=32 (18.75% of tile width). Centered with -3px offset on the perpendicular axis.
