<erase_rule_locked>
**Remove-mode edge erase rule (locked — checker Warning #4 resolution):**

When in remove mode and the cursor is at `(rawGridX, rawGridY)` in tile coordinates (possibly fractional after pointer→grid conversion):

1. Compute the nearest lattice point: `nx = Math.round(rawGridX)`, `ny = Math.round(rawGridY)`.
2. The 4 candidate incident edges of that lattice point are:
   - `(nx - 1, ny, 'h')` — horizontal edge whose right endpoint is the lattice point
   - `(nx,     ny, 'h')` — horizontal edge whose left endpoint is the lattice point
   - `(nx, ny - 1, 'v')` — vertical edge whose bottom endpoint is the lattice point
   - `(nx, ny,     'v')` — vertical edge whose top endpoint is the lattice point
3. For each candidate, compute its midpoint in tile coordinates:
   - `(x, y, 'h')` midpoint = `(x + 0.5, y)`
   - `(x, y, 'v')` midpoint = `(x, y + 0.5)`
4. Pick the candidate whose midpoint has the smallest Euclidean distance to `(rawGridX, rawGridY)`.
5. If that distance is `≤ 0.5` tile units AND an actual PlacedEdge with those coordinates exists in `placedEdges`, remove that edge.
6. Otherwise, no-op for the edge layer (still proceed with tile erase at `(Math.round(rawGridX), Math.round(rawGridY))` if a tile exists there).

Note: only ONE edge per erase sample (the nearest). This avoids mass-deleting incident edges when the cursor is right on a lattice point.
</erase_rule_locked>

      // 本地复现 Plan 03 Task 3 锁定规则，纯函数自洽测试
      function pickIncidentEdgeToErase(
        rawGridX: number,
        rawGridY: number,
        edges: Record<string, { x: number; y: number; orientation: 'h' | 'v' }>,
      ): { x: number; y: number; orientation: 'h' | 'v' } | null {
        const nx = Math.round(rawGridX)
        const ny = Math.round(rawGridY)
        const candidates: Array<{ x: number; y: number; orientation: 'h' | 'v' }> = [
          { x: nx - 1, y: ny, orientation: 'h' },
          { x: nx, y: ny, orientation: 'h' },
          { x: nx, y: ny - 1, orientation: 'v' },
          { x: nx, y: ny, orientation: 'v' },
        ]
        let best: { edge: typeof candidates[number]; dist: number } | null = null
        for (const c of candidates) {
          const mx = c.orientation === 'h' ? c.x + 0.5 : c.x
          const my = c.orientation === 'h' ? c.y : c.y + 0.5
          const dx = mx - rawGridX
          const dy = my - rawGridY
          const dist = Math.sqrt(dx * dx + dy * dy)
          if (best === null || dist < best.dist) {
            best = { edge: c, dist }
          }
        }
        if (!best || best.dist > 0.5) return null
        const exists = Object.values(edges).some(
          (e) => e.x === best!.edge.x && e.y === best!.edge.y && e.orientation === best!.edge.orientation,
        )
        return exists ? best.edge : null

    // 边擦除（Phase 02.1）— 锁定规则
    // 1) 最近格点 = Math.round(rawGridX/Y)
    // 2) 4 条入射候选边（2 水平 + 2 垂直）
    // 3) 各自中点到光标的欧氏距离取最小
    // 4) 仅当距离 ≤ 0.5 且边真实存在时擦除
    // ========================================
    const eraseNearestIncidentEdge = useCallback(
      (rawGridX: number, rawGridY: number): boolean => {
        const nx = Math.round(rawGridX)
        const ny = Math.round(rawGridY)
        const candidates: Array<{ x: number; y: number; orientation: EdgeOrientation }> = [
          { x: nx - 1, y: ny,     orientation: 'h' },
          { x: nx,     y: ny,     orientation: 'h' },
          { x: nx,     y: ny - 1, orientation: 'v' },
          { x: nx,     y: ny,     orientation: 'v' },
        ]
        let best: { c: typeof candidates[number]; dist: number } | null = null
        for (const c of candidates) {
          const mx = c.orientation === 'h' ? c.x + 0.5 : c.x
          const my = c.orientation === 'h' ? c.y : c.y + 0.5
          const dx = mx - rawGridX
          const dy = my - rawGridY
          const dist = Math.sqrt(dx * dx + dy * dy)
          if (best === null || dist < best.dist) best = { c, dist }
        }
        if (!best || best.dist > 0.5) return false

        const state = useEditorStore.getState()
        for (const [id, e] of Object.entries(state.placedEdges)) {
          if (e.x === best.c.x && e.y === best.c.y && e.orientation === best.c.orientation) {
            state.removeEdge(id)
            return true
          }
        }
        return false
      },
      [],
    )
